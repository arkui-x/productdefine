## 配置仓目录结构<a name="section113275517516"></a>

```sh
productdefine/common
├── base                          # 最小系统部件集合
│   └── standard_system.json      # 标准系统最小部件集合
└── products                      # 系统组件形态配置文件，配置文件名称与product name保持一致
    └── arkui-x.json          # SDK部件集合
```
